New Feature Added:

Delete All Button

Added a "Delete All" button in the Key Stakeholders section header

Confirmation Modal

Shows a confirmation dialog before deleting to prevent accidental deletion

Success Notification

Shows how many stakeholders were deleted after confirmation

Benefits

  • • Quickly clear all stakeholders when switching between customers
  • • Makes it easier to save and restore different customer profiles
  • • Prevents having to delete stakeholders one by one
  • • Confirmation dialog prevents accidental deletion

How to Use:

  1. 1. Navigate to the Key Stakeholders section
  2. 2. Click the "Delete All" button in the top right corner
  3. 3. Confirm the deletion in the modal dialog
  4. 4. All stakeholders will be removed and the state will be saved
